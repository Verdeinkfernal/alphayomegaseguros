# Alpha y Omega Seguros — Insurance Team Landing Page

Landing page for Alpha y Omega, an insurance sales team (BMI health & life insurance)
operating within the Eleven agency network in Ecuador.

## Features
- Advisor/agent directory section (asesores)
- Lead capture with confirmation page (gracias.html)
- SEO-ready (sitemap, Open Graph image)
- Deployed via Netlify

## Tech stack
- HTML5, CSS3, JavaScript
- Netlify deployment (netlify.toml)

## Related project
Paired with botpress-ao-seguros (https://github.com/Verdeinkfernal/botpress-ao-seguros),
a chatbot that answers agent questions about the same insurance products.
